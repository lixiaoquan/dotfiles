# Install
## Install basic
```bash
./install.sh
```
## Download tools
```bash
install/download.sh
```
## Install tools
```bash
install/tmux.sh
install/powerline_fonts.sh
install/zsh.sh
install/ohmyzsh.sh
install/fzf.sh
install/node.sh
install/vim.dev.sh
install/git.sh
install/lazygit.sh
install/clangd.sh
...
```

## Troubleshooting
### Build zsh with error about cur_term
apt install libncurses-dev

1. PlugUpdate
2. CocUpdate
3. TSUpdate
